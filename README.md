# website
My first ever website, I want to create an indie website that I would use as a blog to talk about things I love.
I want to create a website with a 2000s vibe, but most importantly I want to create everything by hand, WITHOUT AI at any point.
My main project for this website would be to talk about the music I love, the albums that I discover and the albums that I want to share.
At some point I would also like to have a page dedicated to KCORP, maybe with my favourite moments and pictures of this club, and even share some fan arts.

I have never used html/css so for the moment I am just at the point where I learn everything, I am in the experimental part where I just try to figure out how it works.
